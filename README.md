## このリポジトリについて

個人開発のための複数アプリを、共通基盤の上で育てていくプロジェクトです。
（例：推し活情報集約、推し活に紐づく家計管理、タスク管理 など）

### 重要な方針（北極星）
- **個人利用が前提**（公開/共有するものと、非公開のものを分離する）
- **長期運用で迷わないこと**（設計判断は履歴として残す）
- **可読性と変更容易性を優先**（小さく作って小さく直す）

### 開発とAIの役割分担
- **Claude：設計提案 + 実装**
- **Codex：レビュー + リファクタリング**
